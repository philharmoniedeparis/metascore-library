Send an XMLHttp request

Parameters:

  • url String

    The URL to which the request is sent

  • options Object

    to set for the request

    • [method='GET'] String optional

      The method used for the request (GET, POST, or PUT)

    • [headers={}] Object optional

      An object of additional header key/value pairs to send along with requests

    • [async=true] Boolean optional

      Whether the request is asynchronous or not

    • [data={}] Object optional

      Data to be send along with the request

    • [dataType='json'] String optional

      The type of data expected back from the server

    • [complete] Funtion optional

      A function to be called when the request finishes

    • [success] Funtion optional

      A function to be called if the request succeeds

    • [error] Funtion optional

      A function to be called if the request fails

    • [scope=this] Object optional

      The object to which the scope of the above functions should be set
