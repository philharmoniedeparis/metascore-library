naturalSort

(
  • [insensitive=false]
)
Function static

A natural sort function generator

Parameters:

  • [insensitive=false] Boolean optional

    Whether the sort should not be case-sensitive

Returns:

Function:

The sorting function

Example:

var arr = ["c", "A2", "a1", "d", "b"];
arr.sort(metaScore.Array.naturalSort(true));
// ["a1", "A2", "b", "c", "d"]

var arr = ["c", "A2", "a1", "d", "b"];
arr.sort(metaScore.Array.naturalSort(false));
// ["A2", "a1", "b", "c", "d"]
