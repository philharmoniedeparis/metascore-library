CuePoint Class

Extends Evented
Module: Player

A class for managing media cuepoints to execute actions at specific media times

Constructor

CuePoint

(
  • configs
)

Parameters:

  • configs Object

    Custom configs to override defaults

    • media player.component.Media

      The media component to which the cuepoint is attached

    • [inTime] Number optional

      The time at which the cuepoint starts

    • [outTime] Number optional

      The time at which the cuepoint stops

    • [considerError] Boolean optional

      Whether to estimate and use the error margin in timed events
